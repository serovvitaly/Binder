Binder
======

Документо-ориентированная система управления контентом

# Основные функции
* Мультиязычность
* Мультисайтовость
* Модульность
* Разграничение прав доступа


# Админка
* Управление пользователями и ролями
* Управление структурой
* Управление сайтами
* Управление Шаблонами
* Управление документами
